ObjectTools
===========

java script tool to print object information

example use 
in browser
ObjectTools.setMode('browser');

ObjectTools.printObject(yourObject);

in node

var objectTools = require('./tools.js');

objectTools.printObject(yourObject);

result like as
 a: as
 
 b: 2,
 
 c: 
 
    d: 1
    
    k: 3
